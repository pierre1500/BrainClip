# BrainClip Vault

This folder contains your clipped notes organized by domain.

## Structure

Notes are automatically organized by the source domain:

```
vault/
├── github.com/
│   └── 2025-12-11-interesting-repo.md
├── medium.com/
│   └── 2025-12-11-great-article.md
├── arxiv.org/
│   └── 2025-12-11-research-paper.md
└── ...
```

## Note Format

Each note includes:
- **YAML Frontmatter**: Title, source URL, author, tags, date clipped
- **Summary**: AI-generated summary of the content
- **Key Points**: Main takeaways from the page
- **Notes**: Detailed content with proper formatting

## Syncing with Obsidian

To use this vault with Obsidian:

1. Open Obsidian
2. Click "Open folder as vault"
3. Select the `vault` folder from this project
4. Your clipped notes will appear organized by domain

## Contributing Your Notes

This vault is part of the open-source project! Your research journey becomes part of BrainClip.

When you clip interesting content, consider sharing valuable notes with the community through pull requests.

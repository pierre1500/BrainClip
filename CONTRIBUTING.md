## How to Contribute

### Reporting a Bug

1. Check that the bug hasn't already been reported in [Issues](../../issues)
2. Create a new issue with a clear title and detailed description
3. Include steps to reproduce the bug

### Proposing a Feature

1. Open an issue to discuss your idea
2. Wait for feedback before starting development

### Submitting Code

1. **Fork** the project
2. Create a branch for your feature (`git checkout -b feature/my-feature`)
3. **Commit** your changes (`git commit -m 'Add my feature'`)
4. **Push** to the branch (`git push origin feature/my-feature`)
5. Open a **Pull Request**

## Code Standards

- Write clean and readable code
- Comment your code when necessary
- Test your changes before submitting

## Code of Conduct

Be respectful and kind to other contributors.

## Questions?

Feel free to open an issue for any questions!
